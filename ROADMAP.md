# PerfectUI Roadmap

## 🎯 Vision
Build a comprehensive, modern UI component library for Angular that is:
- Easy to use
- Highly customizable
- Accessible
- Well-documented
- Production-ready
- Tree-shakable

---

## 📦 Current Components (v2.0)

- [x] **perfectui/dialog** - Dialogs, modals, alerts, confirms, prompts
- [x] **perfectui/toastr** - Toast notifications
- [x] **perfectui/otp** - OTP (One-Time Password) input

---

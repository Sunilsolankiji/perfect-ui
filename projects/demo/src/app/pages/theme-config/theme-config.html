How to Use

1. Import the provider

import {{ '{' }} providePerfectUI {{ '}' }} from '@sunilsolankiji/perfectui/core';

2. Configure in your app

providePerfectUI({{ '{' }}
  theme: 'indigo-pink',
  darkMode: 'auto',
  density: 'default'
{{ '}' }})

3. Use ThemeService for runtime changes

themeService.setTheme('deep-purple-amber');
themeService.toggleDarkMode();
themeService.setDensity('compact');

4. Or import a prebuilt CSS theme

@import '@sunilsolankiji/perfectui/themes/indigo-pink.css';

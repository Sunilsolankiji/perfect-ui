  • 🎨 Global Theme — styled entirely with PerfectUI CSS variables
  • 🧱 4 Variants — default, outline, underline, filled
  • 📐 3 Sizes — small, medium, large
  • 🔁 Single & Multi-Select with optional maxSelections
  • 🏷️ Tag overflow — maxVisibleTags caps inline tags with a +N more chip

  • 📝 Forms — ngModel & ReactiveForms via ControlValueAccessor
  • ⚙️ Input-signal config — no global provider required

✨ Features

  • 🎨 Global Theme Support - Automatically uses PerfectUI's global theme colors
  • 📐 5 Size Options - Small, Medium, Large, X-Large, Fullscreen
  • 📍 3 Positions - Center, Top, Bottom
  • 🔔 Built-in Dialogs - Alert, Confirm, Prompt, Danger
  • 📄 Template Support - Pass Angular templates to dialogs
  • 🧩 Custom Components - Use your own components as dialogs
  • ⌨️ Keyboard Support - ESC to close, Enter to confirm
  • 📚 Stacking Support - Multiple dialogs on top of each other
  • ♿ Accessible - Full ARIA support

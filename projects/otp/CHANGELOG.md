# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-20

### Added
- Initial release of @perfectui/otp
- OTP input component with configurable length (default 6 digits)
- Support for numeric, alphanumeric, and custom input types
- Auto-focus on next input after entry
- Paste support for full OTP codes
- Backspace navigation to previous input
- Form integration with ControlValueAccessor
- Reactive forms and template-driven forms support
- Customizable styling with CSS variables
- Multiple themes: default, outline, underline, filled
- Size variants: small, medium, large
- Disabled and readonly states
- Auto-submit on complete option
- Masked input option for security
- Accessible with proper ARIA attributes
- Keyboard navigation support
- Error state styling
- Success state styling
- Animation effects
